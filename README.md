# Weather Dashboard Backend

## Documentation API (Swagger)

La documentation interactive de l’API est disponible ici (lorsque l’application tourne en local) :

[http://localhost:8080/swagger-ui](http://localhost:8080/swagger-ui)

---

- Lancez l’application avec `mvn spring-boot:run` ou via Docker.
- Accédez à l’URL ci-dessus pour explorer et tester tous les endpoints REST.

## À propos

Backend Java (Spring Boot) pour l’application Weather Dashboard.
- API REST connectée à OpenWeatherMap
- Gestion des villes favorites
- Documentation OpenAPI/Swagger intégrée
